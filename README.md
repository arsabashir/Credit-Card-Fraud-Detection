# Credit-Card-Fraud-Detection

Detecting fraudulent credit card transactions using machine learning.

## Dataset
Kaggle Credit Card Fraud Detection Dataset

## Approach
- Exploratory Data Analysis
- Data preprocessing and handling class imbalance
- Classification model training and evaluation

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Google Colab

## Results
Accuracy on Training data :  0.9708333333333333
Accuracy score on Test Data :  0.9666666666666667
